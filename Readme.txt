QuadMoad - a quad pickup sound mod for Warfork

installation on Windows:

1. Copy the file data0_21pure.pk3 located in the folder:
/.../Steam/steamapps/common/fvi/Warfork.app/Contents/Resources/basewf
to the folder:
/.../Documents/My Games/Warfork 2.1/basewf

2. Open the *copied* data0_21pure.pk3 in 7zip
3. Open the QuadMoad archive in 7zip

4. Copy the sound folder from the QuadMoad archive into data0_21pure.pk3

5. When prompted if you are sure you want to copy files to archive, press yes
